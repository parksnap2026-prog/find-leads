# find-leads

Lead finder and outreach app (Flask legacy + Next.js web app in `web/`).

## Vercel

Import [parksnap2026-prog/find-leads](https://github.com/parksnap2026-prog/find-leads) and set:

| Setting | Value |
|---------|-------|
| **Root Directory** | `web` |
| **Framework** | Next.js |

Config lives in [`web/vercel.json`](web/vercel.json). See [`web/DEPLOY.md`](web/DEPLOY.md) for env vars and the full checklist.
